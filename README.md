# webmoive
1.思路先从静态网页中将页面抽离出来,随后在一步一步的抽离成为组件
2.这里的布局不要随意使用p标签,会有初始样式的存在,不要与已知的标签重名;
3.如果在app.vue中设置的样式会优先被子组件中的覆盖;
